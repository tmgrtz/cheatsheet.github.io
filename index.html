Command Line Commands
Cheat Sheet

In this cheat sheet you will find basic command line commands for navigating between directories and creating new directories and files.

Navigation

command name description
pwd print working directory prints the name of the working directory
ls list lists all files and directories in the working directory
cd change directories takes a directory name as an argument, and switches into that directory

Creating, copying, moving, removing

command name description
mkdir make directory takes in a directory name as an argument, and then creates a new directory in the current working directory
touch touch It takes in a file name as an argument, and then creates a new empty file in the current working directory
rm deletes files Command removes a file from the file system
rm -r Command deletes a directory and all of its child directories.
